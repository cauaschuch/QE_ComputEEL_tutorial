Exemplo: Ac₂CdGe (Grupo espacial 225 - Fm-3m)

OBS: Existem 230 grupos espaciais que combinam simetria pontual com simetrias translacionais.

Para o composto Ac₂CdGe com grupo espacial 225, temos:

  • 4a Ge → (0, 0, 0)
  • 4b Cd → (1/2, 0, 0)
  • 8c Ac → (1/4, 3/4, 3/4)

Consultando o site Bilbao Crystallographic Server (WYCKPOS para o número 225), obtemos as posições equivalentes:

(0, 0, 0)
(0, 1/2, 1/2)
(1/2, 0, 1/2)
(1/2, 1/2, 0)

Exemplos de posições obtidas aplicando as simetrias:

  • (0, 1/2, 1/2) + (1/4, 1/4, 1/4) = (1/4, 3/4, 3/4)

Posições finais em coordenadas fracionárias (em alat):

0.00000000  0.00000000  0.00000000  Ge
0.50000000  0.50000000  0.50000000  Cd
0.75000000  0.75000000  0.75000000  Ac
0.25000000  0.25000000  0.25000000  Ac

Parâmetro de rede:

  • Como é uma estrutura cúbica, usamos ibrav = 2 (cúbica F - face centrada).
  • celldm(1) = 7.95 Å = 7.95 × 1.8897259886 = 15.0233 Bohr

Vetores de rede (ibrav = 2):

v1 = (a/2) * (-1, 0, 1)
v2 = (a/2) * ( 0, 1, 1)
v3 = (a/2) * (-1, 1, 0)

Essas informações permitem montar o arquivo de entrada para simulações no Quantum ESPRESSO com coordenadas em crystal e unidade de rede em alat.
